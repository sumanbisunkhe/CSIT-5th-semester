CSS Styling Methods

1. Inline CSS

Inline CSS applies styles directly to individual elements using the style attribute.

2. Internal CSS

Internal CSS is defined within the HTML document's <style> tag in the <head> section.

3. External CSS

External CSS is defined in a separate .css file and linked to the HTML document.
